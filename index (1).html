Connect to Google Sheets

This system uses Google Apps Script as its backend. All data is stored in your Google Sheet and synced every 10 seconds across all users in real time.

Setup steps:
  1. Open your Google Sheet (or create a new one in xavier22ramos@gmail.com)
  2. Go to Extensions → Apps Script
  3. Paste the contents of KeyMonitoring_GAS_Backend.js and save
  4. Click Deploy → New Deployment → Web App
  5. Set: Execute as = Me, Who has access = Anyone
  6. Copy the Web App URL and paste below
